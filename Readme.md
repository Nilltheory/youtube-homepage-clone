# YouTube Homepage Clone

A responsive clone of the YouTube homepage built using **HTML5** and **CSS3**.

This project was created while learning front-end web development by following SuperSimpleDev's HTML & CSS course. Along the way, I explored CSS Grid, Flexbox, positioning, media queries, and responsive layouts while also fixing layout issues on my own to better understand how modern web pages are built.

---

## Features

* Responsive video grid
* Fixed header
* Fixed sidebar
* Search bar with icons
* Hover tooltips
* Notification badge
* Video cards with thumbnails and metadata
* CSS Grid & Flexbox layouts
* Media queries for different screen sizes

---

## Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries
